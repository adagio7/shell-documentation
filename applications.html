applications.unsafe_decorator module

class applications.unsafe_decorator.UnsafeApp(app: App)[source]

Bases: App

Designate an App as unsafe, causing it to output any error messages.

An unsafe version of an app has the same semantics as the original app, but instead of raising exceptions, it prints the error to its stdout. This prevent long sequences from terminating early when some intermediate commands fail.

Application of the Decorator Design Pattern.

Raises:
None
