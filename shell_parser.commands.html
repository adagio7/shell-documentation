exec(inp=None)[source]
Expand arguments and execute the specified App, performing redirections.
The order of operations is as follows:
1. Perform command substitution on args, potentially producing new arguments
2. Perform filename expansion on the arguments
3. Perform redirections
4. Execute the application

shell_parser.commands.seq module

class shell_parser.commands.seq.Seq(left: Command, right: Command)[source]

Bases: Command

Delimited by a semi-colon (;).
Denotes a sequence of commands to be executed.
The commands are run left to right,
and if an exception is thrown while running the left,
the right will not be executed.
** Note this is where unsafe commands can be helpful. **
Raises:

None

exec(inp=None)[source]
Piped input is supplied only to the first (left) Command.
If execution result of both commands is not None,
return the results joined by a newline.
Otherwise, return the execution result that is not None.
If both execution results are None, return None
